# MS.Tech Agency Landing Page PRD

## Overview
A modern, high-performance agency landing page for Morning Star Technology (MS.Tech).

## Features
- **Hero Section**: High-impact introduction with branding and Call-to-Action.
- **Service Cards**: Detailed showcase of agency services.
- **Portfolio**: Display of past work and successful projects.
- **Contact Form**: Direct communication channel for potential clients.
- **Responsive Layout**: Seamless experience across mobile, tablet, and desktop.
- **Animation**: Immersive UI using Framer Motion or similar.

## Tech Stack
- Frontend: Next.js 16+, React 19+, TypeScript, Tailwind CSS v4.
- Infrastructure: Supabase (for form data).
